# Bike-sales-dashboard
Interactive Excel dashboard project analyzing bike sales data using data cleaning, pivot tables, and visualization techniques. Includes a full end‑to‑end workflow: raw data → cleaned dataset → pivot table analysis → final dashboard with slicers. Ideal for demonstrating practical Excel analytics and business intelligence skills.

## Project Overview
This project showcases a full Excel analytics workflow:

--> Data Cleaning

--> Pivot Table Creation

--> Interactive Dashboard Design

The final dashboard allows users to explore bike purchase patterns using slicers for Marital Status, Region, and Education Level.

## File Structure
The Excel workbook contains four organized sheets:

--> Raw_Data — original dataset, untouched

--> Cleaned_Data — standardized and cleaned data

--> Pivot_Tables — all pivot tables used for the dashboard

--> Dashboard — final interactive dashboard

This structure follows professional BI and analytics standards.

## Data Cleaning Steps
Key cleaning operations performed:

--> Removed duplicates

--> Standardized categorical values

--> Created age brackets

--> Cleaned commute distance values

--> Checked for missing values

The cleaned dataset was then used to build pivot tables and visuals.

## Pivot Table Analysis
Pivot tables were created to analyze:

--> Average income per purchase

--> Bike purchases by age bracket

--> Bike purchases by commute distance

These pivot tables feed directly into the dashboard visuals.

## Dashboard Features
The dashboard includes:

--> Interactive slicers (Marital Status, Region, Education)

--> Bar chart: Average income per purchase

--> Line chart: Purchases by age bracket

--> Line chart: Purchases by commute distance

The design focuses on clarity, simplicity, and insight.

## How to Use

--> Open the Excel file

--> Go to the Dashboard sheet

--> Use the slicers to filter the visuals

--> Explore insights across demographics and regions

Key Insights
Some insights revealed by the dashboard:

--> Married customers show different purchasing patterns than single customers

--> Middle‑aged customers are more likely to purchase bikes

--> Customers with shorter commutes tend to buy more bikes
